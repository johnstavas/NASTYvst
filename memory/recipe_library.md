# Recipe Library — Vintage Audio Archetype Compositions

**Purpose.** This is the "how to build it" book that pairs with the
ops catalog. The catalog answers *what tools do I have* — this answers
*how do I combine those tools to build a Marshall JTM45 / Fairchild 670
/ EMT 250 / etc.*

**Source.** Compositions distilled from Gemini Deep Research output
(2026-04-27, "Audio DSP Catalog Research Brief — TIGHTENED" prompt),
cross-validated against Perplexity Deep Research where citations
overlap. Each recipe lists its primary source so the AI agent can cite
authoritative documents when generating a graph.

**Format per recipe.** Five fields:
1. **Canonical signal chain** — plain-English signal flow (the way an
   engineer would describe it on a napkin)
2. **Op composition** — using our catalog vocabulary; the AI agent
   builds graph.json from this
3. **Primary source** — schematic, service manual, or AES paper backing
   the topology
4. **Key character notes** — sound-designer-voice description of what
   defines this archetype's signature
5. **Notes** — implementation gotchas, missing ops if any, etc.

**ADDITIVE-ONLY discipline.** This file does not modify the catalog or
locked Tier-S sources. It's a parallel reference doc consulted at recipe
authoring time.

## TUBE GUITAR AMPS

### 1. Marshall JTM45 (British vintage tube amp)

**Canonical signal chain.** input → triodeStage (V1) → mix (Normal/High Treble channels) → cathodeBiasShift (V2 DC-coupled to tone stack) → baxandallTone (Marshall tone stack) → phaseInverter (Long-Tailed Pair) → pushPullPower (KT66/5881) → xformerSat (Output Transformer) → tubeRectifierSag (GZ34) → speaker

**Op composition.** `triodeStage`, `mix`, `cathodeBiasShift`, `baxandallTone`, `phaseInverter`, `pushPullPower`, `xformerSat`, `tubeRectifierSag`

**Primary source.** Marshall JTM45 Reissue Circuit Diagram (DWG.No. JTM45.DGM). Cross-ref: Aiken Amplifiers, drtube.com/marshall-jtm45/, Doug Hoffman amp schematics (el34world.com).

**Key character notes.** The defining British crunch originates largely from the V2 cathode follower driving the tone stack. Because it's DC-coupled, heavy signals push the grid positive relative to the cathode, causing grid-current clipping that thickens the upper midrange and compresses transients dynamically. The dual KT66 power section coupled with a GZ34 tube rectifier introduces significant power-supply voltage sag under heavy transient loads — a distinct dynamic bloom, loose spongy low-end, highly interactive playing feel. Output transformer saturation introduces odd-order harmonics adding aggressive "bite" before the speaker limits the high frequencies.

**Notes.** Requires `cathodeBiasShift` (#157) — currently ⬜. Until built, fall back to a `triodeStage` chained into a soft-knee shelf.

### 2. Vox AC30 (British chime amp)

**Canonical signal chain.** input → triodeStage (ECC83) → mix (Top Boost) → baxandallTone (Interactive Treble/Bass) → phaseInverter (Long-Tailed Pair) → discreteClassAStage (Cathode-biased EL84s) → pushPullPower (Quad EL84) → xformerSat → speaker

**Op composition.** `triodeStage`, `mix`, `baxandallTone`, `phaseInverter`, `discreteClassAStage`, `pushPullPower`, `xformerSat`

**Primary source.** Vox AC30TB Standard 1989 Schematic / JMI 1960 OS/065. Cross-ref: voxshowroom.com, thetubestore.com.

**Key character notes.** The signature "chime" is fundamentally generated by the total absence of global negative feedback in the power section. This absence allows the output transformer and the loudspeakers to interact dynamically, emphasizing the resonant peaks of the speaker cones. The power section operates four EL84 pentodes in a hot cathode-biased Class AB configuration (often colloquially but incorrectly termed Class A). When pushed, the phase inverter clips asymmetrically before the power tubes, yielding an aggressive upper-midrange emphasis that cuts through dense mixes effortlessly.

### 3. Fender Twin Reverb (American clean amp)

**Canonical signal chain.** input → triodeStage (12AX7) → baxandallTone (T/M/B) → triodeStage (Gain) → mix (Dry/Reverb) → spring (Accutronics Tank) → phaseInverter (12AT7) → pushPullPower (Quad 6L6GC) → xformerSat → speaker

**Op composition.** `triodeStage`, `baxandallTone`, `mix`, `spring`, `phaseInverter`, `pushPullPower`, `xformerSat`

**Primary source.** Fender AA763 / AB763 Schematic. Cross-ref: schematicheaven.net/fenderamps/fender_deluxe_aa763.pdf, robrobinette.com.

**Key character notes.** Renowned for massive headroom and pristine American clean tones, the circuit relies on a fixed-bias quad 6L6GC power section that delivers tight, unyielding low frequencies. The tone stack imposes a drastic mid-frequency scoop due to its specific capacitor values, loading down the signal significantly. When combined with the phase shift of the recovery triode and the bright capacitor networks situated across the volume potentiometers, this produces the trademark "Fender shimmer" and crystalline high-end clarity. The phase inverter utilizes a 12AT7 (lower gain but higher current capability than 12AX7), ensuring power tubes are driven cleanly right up to the point of catastrophic clipping.

### 4. Fender Tweed / Bassman 5F6-A (American breakup)

**Canonical signal chain.** input → triodeStage (12AY7) → mix → triodeStage (12AX7 DC-coupled) → baxandallTone (Tweed Tone Stack) → phaseInverter (Long-Tailed Pair) → pushPullPower (Dual 5881) → xformerSat → tubeRectifierSag (GZ34) → speaker

**Op composition.** `triodeStage`, `mix`, `baxandallTone`, `phaseInverter`, `pushPullPower`, `xformerSat`, `tubeRectifierSag`

**Primary source.** Fender Bassman 5F6-A Schematic. Cross-ref: pureadmin.qub.ac.uk Wave Digital Filter Case Study, ampbooks.com classic-circuits.

**Key character notes.** The 5F6-A is the direct genetic predecessor to almost all modern high-gain rock amplifiers. Its character is defined by a highly interactive tone stack driven by a hot-biased, DC-coupled cathode follower. This unique staging drives the long-tailed pair phase inverter exceptionally hard, causing asymmetric clipping that translates to a throaty, raw upper-midrange breakup. The dynamic squish of the GZ34 tube rectifier compresses hard transients, yielding a highly touch-sensitive response. The 12AY7 input tube provides a lower gain structure at the very front, allowing the player to push the volume controls higher and generate distortion deeper within the amplifier's architecture.

## HI-FI TUBE AMPS

### 5. Mullard 5-20 / Quad II / Leak Stereo 20 (Hi-fi British tube)

**Canonical signal chain.** input → bjtSingleStage (EF86 Pentode) → phaseInverter (ECC83 direct-coupled) → pushPullPower (EL34 / KT66 distributed load) → ultraLinearScreen → xformerSat → tubeRectifierSag

**Op composition.** `bjtSingleStage` (EF86 voltage amp role), `triodeStage`, `phaseInverter`, `pushPullPower`, `ultraLinearScreen`, `xformerSat`, `tubeRectifierSag`

**Primary source.** Mullard 5-valve 20-watt High-Quality Amplifier Design (1955) / Leak Stereo 20 Schematic. Cross-ref: Mullard 1963 *Circuits for Audio Amplifiers* (locked Tier-S), keith-snook.info/schematic/QUAD-II-Schematic.pdf.

**Key character notes.** These topologies represent the absolute pinnacle of mid-century British high-fidelity audio engineering. They rely on ultra-linear distributed screen loading, where the power tube screen grids are fed from specific percentage taps on the primary winding of the output transformer. This topology yields the low distortion and low output impedance of a triode, coupled with the high efficiency of a pentode. Sonically, it is extraordinarily linear, presenting a deep, uncolored low end and a silky, non-fatiguing high-frequency response. Direct coupling between the EF86 voltage amplifier and the phase splitter minimizes low-frequency phase shifts, ensuring absolute stability.

### 6. Williamson amp (1947 reference design)

**Canonical signal chain.** input → triodeStage → phaseInverter (Concertina) → triodeStage (Driver) → pushPullPower (KT66 triode-connected) → xformerSat (high-quality multisection OT)

**Op composition.** `triodeStage`, `phaseInverter`, `pushPullPower`, `xformerSat`

**Primary source.** Williamson, D.T.N., "Design For A High Quality Amplifier," *Wireless World*, April + May 1947. Open PDFs at worldradiohistory.com (Wireless-World-1947-04.pdf and -05.pdf), introni.it/pdf/The Williamson Amplifier.pdf, dalmura.com.au.

**Key character notes.** The sonic footprint of the Williamson amplifier is defined by absolute linearity and a massive reserve of phase margin. This is afforded by wrapping 20 dB of global negative feedback around an exceptionally high-quality output transformer. The triode-strapped KT66 tubes deliver a pristine, almost analytical clarity across the full audio spectrum. The concertina phase splitter ensures mathematically perfect AC balance, meaning that even-order harmonics are canceled almost entirely in the output stage, leaving a sound that is incredibly clean with virtually nonexistent crossover distortion. The stringent transformer requirements prevent the high-frequency ringing typical of lesser feedback designs.

## SPRING & PLATE REVERBS

### 7. Hammond B-3 + spring tank

**Canonical signal chain.** input (Tonewheels) → sineOsc (Additive Drawbars) → mix → triodeStage (AO-28 Preamp) → spring (Necklace delay tank) → mix → xformerSat

**Op composition.** `sineOsc`, `mix`, `triodeStage`, `spring`, `xformerSat`

**Primary source.** Hammond Organ B3/C3 Service Manual. Cross-ref: synthmania.com/Hammond-Organ-B3-C3-Service-Manual.pdf, hammondorganco.com.

**Key character notes.** The mechanical tonewheels generate slightly impure sine waves that contain significant mechanical noise and crosstalk leakage. When combined additively via the drawbars, they create a dense harmonic structure heavily colored by key-click transients caused by physical switch bouncing. The internal AO-28 tube preamp saturates the sub-harmonics beautifully, adding low-midrange girth. The necklace spring reverb introduces a dark, sloshy mechanical reflection that blends intimately with the direct signal, heavily masking the high-frequency purity and imparting a haunting, acoustic spatial dimension.

### 8. EMT 140 plate reverb

**Canonical signal chain.** input → sidechainHPF (Bass Cut filter) → softLimit (Driver overload protection) → plate (Steel Sheet) → gain

**Op composition.** `sidechainHPF`, `softLimit`, `plate`, `gain`

**Primary source.** EMT 140 TS Reverberation Unit Technical Manual. Cross-ref: bbceng.info/ti/eqpt/EMT140.pdf, preservationsound.com/wp-content/uploads/2013/09/EMT_140_ts_1971.pdf, technicalaudio.com.

**Key character notes.** The EMT 140 creates exceptionally dense, bright, and immediate reflections due to the high-speed propagation of bending waves through a massive, tensioned steel plate. A built-in peak compressor on the driver limits transient splat, preventing the mechanical transducers from bottoming out. The result is a smooth, metallic decay that sits perfectly in a mix without muddying the fundamental frequencies of the dry signal. The decay time is physically manipulated by a damping pad; mathematically, the high frequencies decay faster than the lows, requiring the high-pass input filter to maintain clarity.

### 9. EMT 250 digital reverb

**Canonical signal chain.** input → oversample2x → aliasingDecimator (12-bit / 32.75 kHz) → schroederChain (Comb/Allpass array) → fpDacRipple → lfo (Modulation) → mix

**Op composition.** `aliasingDecimator`, `schroederChain`, `allpass`, `comb`, `lfo`, `fpDacRipple`, `mix`

**Primary source.** EMT 250 Electronic Reverberator Manual / Courier 1976. Cross-ref: tile.loc.gov EMT 250 Courier (July 1985), davidmorrin.com/emt250.html.

**Key character notes.** The EMT 250 is distinguished by its primitive 32.75 kHz sample rate and heavy hardware anti-aliasing filtering, yielding a restricted but intensely warm frequency response. The algorithmic architecture relies on low-density Schroeder topologies combined with slow, wandering LFO modulation on the delay lines. This produces a lush, chorused tail that heavily colors the signal with a distinct, grainy, low-resolution digital signature. The 12-bit conversion introduces a highly musical quantization noise that becomes perceptually prominent as the reverb tail decays into silence. **This is where #182 blesserReverbCore lives** — Blesser's specific scattering topology distinct from #20 fdnCore.

**Notes.** Recipe uses `schroederChain` (#22 — early Schroeder structure) plus optional `blesserReverbCore` (#182 ⬜) when that op ships.

### 10. Lexicon 224 / 480L digital reverb

**Canonical signal chain.** input → aliasingDecimator → fdnCore (Feedback Delay Networks) → allpass (Diffusion) → lfo (Chorusing) → fpDacRipple → mix

**Op composition.** `aliasingDecimator`, `fdnCore`, `allpass`, `lfo`, `fpDacRipple`, `mix`

**Primary source.** Lexicon 224 Service Manual. Cross-ref: therackofages.com/Lexicon/224x digital reverborator/Lexicon-224X-Service-Manual.pdf, archive.org Lexicon 224 manual.

**Key character notes.** Utilizing early fixed-point arithmetic and floating-point converters, the Lexicon 224 derives its spatial magic from complex feedback delay networks (FDNs) heavily modulated by multi-phase LFOs. This modulation prevents the discrete delay lines from ringing at resonant metallic frequencies. The result is an expansive, synthetic, and highly diffuse wash of sound that decays organically. It is characterized by a dark digital grittiness at the tail's end due to quantization noise and early converter inaccuracies.

### 11. AKG BX-20 / BX-25 spring reverb

**Canonical signal chain.** input → phaseInverter (Motional Feedback) → spring (Torsional transmission line) → gain

**Op composition.** `phaseInverter`, `spring`, `gain`

**Primary source.** AKG BX 20E Reverberation Unit Service Manual. Cross-ref: tile.loc.gov AKG BX 20 E Reverberation Unit, barryrudolph.com/recall/manuals/akgbx20.pdf, scribd.com/document/534947622/Akg-Bx20e-Reverb.

**Key character notes.** Unlike standard guitar amplifier spring tanks, the BX-20 utilizes exceedingly long, heavily damped torsional springs encased in a tubular pendulum. This mechanical isolation eradicates the typical "boing" and "splat" associated with spring reverbs. A highly complex motional feedback circuit injects out-of-phase signals into the drive coils to linearize the mechanical response of the springs. This yields an incredibly dark, dense, and plate-like acoustic tail with an extraordinarily smooth high-frequency roll-off.

### 12. Roland Space Echo / Echoplex / Watkins Copicat (tape echo)

## DYNAMICS

### 14. SSL G-series bus comp / dbx 165A (VCA bus comp)

**Canonical signal chain.** input → bjtSingleStage (NE5534) → sidechainHPF → detector (True Peak / RMS) → blackmerVCA (Gain Reduction) → gainComputer → bjtSingleStage

**Op composition.** `bjtSingleStage`, `sidechainHPF`, `detector`, `blackmerVCA`, `gainComputer`

**Primary source.** SSL 500 Series G Comp Manual / dbx 165A Manual & Schematic. Cross-ref: gyraf.dk/gy_pd/ssl/ssl.htm, solid-state-logic.co.jp/docs/XLogic_G-Comp.pdf.

**Key character notes.** Renowned for mix-bus "glue" and aggressive transient punch, these VCA compressors rely on a dominant sidechain architecture where a true-peak or RMS detector drives an extremely fast voltage-controlled amplifier (VCA). The result is a highly responsive, "snappy" compression characteristic that aggressively clamps down on drum transients while retaining mid-range clarity. Engaging a high-pass filter (sidechainHPF) in the detector path prevents heavy bass frequencies from modulating the entire mix, eliminating unwanted pumping artifacts.

## EQUALIZERS / PREAMPS

### 19. Pultec EQP-1A / MEQ-5 (passive program EQ)

**Canonical signal chain.** input → transformerSim → inductorEQ (Passive LC Filter Network) → triodeStage (Make-up Gain) → pushPullPower (Push-pull output) → transformerSim

**Op composition.** `transformerSim`, `inductorEQ`, `triodeStage`, `pushPullPower`

**Primary source.** Pultec EQP-1A manual (locked Tier-S) / MEQ-5 Service Manual at funkwerkes.com. Cross-ref: gyraf.dk/wp-content/uploads/2018/05/Do-A-Pultec.pdf.

**Key character notes.** The passive LC networks allow simultaneous boosting and cutting at the same target frequency. Due to slight phase shifts and bandwidth discrepancies between the boost and cut filter paths, this action creates a unique resonant shelf (the "Pultec trick") that simultaneously adds massive low-end weight while clearing out upper-bass mud. The passive section incurs an 18 dB insertion loss; the tube makeup stage restores this level while adding subtle harmonic bloom, depth, and three-dimensional spatial characteristics.

## SPECIAL / VOCAL / SYNTH

### 27. Heil Talkbox / Roger Troutman / EMS Vocoder 2000 (vocoder/talkbox)

**Canonical signal chain.** input (Modulator) → envelope → filter (Bandpass Analysis Bank) → filter (Bandpass Synthesis Bank) → mix

**Op composition.** `envelope`, `detector`, `filter`, `mix`

**Primary source.** Standard vocoder topologies (general DSP reference).

**Key character notes.** While a mechanical talkbox uses physical acoustics via a plastic tube, a classic analog vocoder (like the EMS 2000) employs parallel bandpass filters. Envelope followers detect the amplitude of the human voice across specific frequency bands and impose those dynamic control voltages onto identical filters processing an external synthesizer signal (the carrier). This creates the robotic, harmonically dense "talking synthesizer" effect characterized by sharp, intelligible vowel formants.

### 28. LinnDrum / E-mu SP-1200 / TR-808 (vintage drum machine)

**Canonical signal chain (SP-1200).** input → samplerNonlinearity (12-bit / 26.04 kHz) → aliasingDecimator (Drop-sample pitch shift) → fpDacRipple → filter (SSM2044)

**Canonical signal chain (TR-808).** trigger → envelope → schmittTriggerOsc / bridgedTeeEQ / noise → discreteClassAStage (VCA)

**Op composition (SP-1200).** `samplerNonlinearity`, `aliasingDecimator`, `fpDacRipple`, `filter`

**Op composition (TR-808).** `trigger`, `envelope`, `schmittTriggerOsc`, `bridgedTeeEQ`, `noise`, `discreteClassAStage`

**Primary source.** TR-808 Service Manual / E-mu SP-1200 Analysis. Cross-ref: pureadmin.qub.ac.uk/ws/portalfiles/portal/125044847/tr_808_cymbal_a_physically_informed_circuit_bendable_digital.pdf, baratatronix.com/blog/808-tom-synthesis, muzines.co.uk/articles/e-mu-sp1200/2517.

**Key character notes.** The SP-1200 imparts a uniquely gritty, dark character derived from its 12-bit, 26.04 kHz conversion and primitive drop-sample pitch shifting. Tuning a sample down introduces intense foldover aliasing and digital artifacting, which is subsequently smoothed out by dark analog SSM2044 filters. Conversely, the TR-808 generates its massive sub-bass and distinct percussion entirely via analog circuits, utilizing ringing bridged-T filters that act as sine wave resonators, and Schmitt trigger oscillators that create incredibly sharp, purely electronic transients.

### 29. Buchla 200e (West-Coast modular: complex osc + LPG)

**Canonical signal chain.** input → complexOsc (FM generation) → wavefolder (Timbral shaping) → vactrolLPG (Low Pass Gate) → envelope

**Op composition.** `complexOsc`, `wavefolder`, `vactrolLPG`, `envelope`

**Primary source.** Buchla 200e Users Guide v1.4. Cross-ref: buchla.com/guides/200e_Users_Guide_v1.4.pdf.

**Key character notes.** Defining the "West-Coast" synthesis paradigm, the Buchla architecture relies on generating harmonic complexity through frequency modulation and aggressive wavefolding rather than subtractive filtering. The transients are shaped by vactrol-based Low Pass Gates (LPG), which simultaneously act as a VCA and a low-pass filter. The slow decay of the photoresistors within the vactrol imparts a sluggish, highly organic, and percussive "bongo-like" ringing that heavily mimics acoustic instrument envelopes.

### 30. Moog modular (East-Coast: VCA + ladder + envelope)

**Canonical signal chain.** input → sineOsc / sawtooth → ladder (Transistor Ladder Filter) → discreteClassAStage (VCA) → envelope (ADSR)

**Op composition.** `sineOsc`, `ladder`, `discreteClassAStage`, `envelope`

**Primary source.** Moog Ladder documentation / Synthesizer Build schematics. Cross-ref: eddybergman.com/2019/12/synthesizer-build-part-7-moog-ladder.html.

**Key character notes.** The archetypal "East-Coast" subtractive sound relies heavily on the patented 24 dB/octave transistor ladder filter. As the filter is driven harder by the oscillators, the transistors saturate symmetrically, creating a warm, thick, and liquid overdrive. High resonance settings cause a distinct drop in low-frequency energy due to feedback phase cancellation, resulting in the iconic, tearing squelch that defines classic analog bass lines and lead patches.

## Recipe #31 — EMI TG12345 Mk2 Channel Strip (Abbey Road, post-1968)

**Era / use.** 1970+ EMI Studios, Hayes / Abbey Road. Anchors recipes
for: late-Beatles (*Abbey Road*, *Let It Be*), Pink Floyd (*Dark Side
of the Moon*, *Wish You Were Here*), Wings, ELO, Roxy Music, Kate Bush
early albums, plus countless 70s British records tracked at Abbey
Road / Air / Olympic.

**Canonical signal chain (per channel):**

```
mic input (Tuchel 5-pin balanced, 200 Ω source)
  → input transformer (1:3.16 step-up, screened)
  → 50 V phantom inject (NOT 48 V — Abbey Road quirk)
  → coarse atten (12 × 5 dB)
  → mic preamp (Amp D · 25 dB ±5 dB · BC109 × 3 · ±20 V rail)
  → channel BPF (30 Hz HP / 20 kHz LP · AOT-trimmed)
  → comp/lim (Out/Compress/Limit · 6 fixed recoveries · gang-detect)
  → presence EQ (8 freqs × ±10 dB)
  → "bass" EQ (centred 500 Hz, ±10 dB)
  → BE740 bass-cut (FLAT / 75 / 150 / 300 / 600 / 1200 Hz HPF)
  → quadrant fader (5 dB loss at standard level)
  → makeup amp (+5 dB · Amp E or Amp K phase-reverse)
  → pan-pot (11 kΩ × 2 reverse-connected)
  → V-to-I converter (Amp B · 20 kΩ output Z)
  → virtual-earth summing buss (Amp C · 20 Ω input Z)
  → output transformer (1:1.77 BIBA · +5 dB)
  → tape send (−10 dBV nominal)
```

**Op composition (catalog vocabulary):**

```
xformerSat[preset='emiInputXfmr']
  → tg12345MicAmp (#202)
  → abbeyChannelBPF (#200)
  → tg12345CompLim (#201) [recovery=0.5s, hold=mid, mode='Compress']
  → abbeyPresenceEQ (#197) [freq=2.8kHz, gain=+4dB]
  → abbey500HzBass (#198) [gain=+4dB]
  → be740BassCut (#199) [cut=75Hz]
  → germaniumBuffer (#203) [preset='abbeyMakeup', gain=+5 dB]
  → xformerSat[preset='tg12345OutputXfmr']
  → virtualEarthBuss (#196)
```

**The germanium thread.** The TG12345 routes every post-fader signal
through a silicon-input / germanium-output buffer (Amp E for makeup,
Amp G for solo/cue, Amp K for phase-reverse). This is a deliberate
EMI design choice — silicon would have been simpler in 1970, but the
germanium ACY21 output stage gives a softer V_BE knee (~0.25 V vs
silicon's ~0.65 V) and a warmer 2H/3H balance at the clipping edge.
**Recipe #31 must include `germaniumBuffer` to match the era.** Without
it, the chain sounds technically correct but missing the cumulative
warmth that defines Abbey Road records.

**Primary source.** EMI Research Laboratories, *EMI TG12345 Mk2
Recording Console Service Handbook* (drawings dated 1970). PDF at
`docs/primary_sources/abbey_road/EMI_TG12345_Mk2_Service_Handbook.pdf`.
Canon file: `memory/emi_tg12345_console.md`.

**Key character notes.**

- **Virtual-earth buss** = clean blooming when many channels add up.
  20-channel mix has only 0.2 dB total cumulative gain shift —
  imperceptible. This is why TG12345 mixes "open up" rather than
  "thicken" as you add channels.
- **500 Hz "bass"** is a low-mid lift, NOT a low shelf. Boosting
  "bass" adds energy from lower midrange up to 500 Hz — the
  warm-but-not-sub Abbey Road tonal fingerprint.
- **8-freq presence EQ** with no sub-500 Hz options forces engineers
  to handle bass via the separate "bass" + the BE740 highpass —
  different mental model from API/Neve combined-band EQs.
- **6 fixed comp recoveries** (0.1/0.25/0.5/1/2/5 s). Most engineers
  ran position 3 (0.5 s) — the "on the kit" pump time heard on most
  TG12345 records.
- **50 V phantom** vs industry 48 V → marginally hotter condenser-mic
  output and slightly different transient handling.
- **BC109 + ±20 V rail** preamp ceiling at +20 dBV → soft-asymmetric
  clipping distinct from the +24 dBV / 4×IC era (SSL 4000, Neve V)
  that followed.

**Notes for the agent.**

- For "Beatles vocal" / "Lennon vocal" prompts: this strip, comp at
  `Compress`, recovery=0.5 s, presence boosted at 2.8 kHz (+4 dB),
  bass-cut at 75 Hz, "bass" boost +2 dB.
- For "Floyd-style spacious": light comp at 1 s recovery, presence
  6.5 kHz (+2 dB), "bass" flat, BE740 at FLAT, then chain into a
  plate reverb (recipe #11 EMT 140) or digital (recipe #14 Lexicon
  224).
- For "snare from *Dark Side*": Compress mode, recovery position 2
  (0.25 s), presence 4.2 kHz (+6 dB), bass-cut 150 Hz.

**Open ingestion debt.** All schematics for #197/198/200/201/203 now
schematic-locked from the Germanium-folder ingestion (2026-04-28 third
+ fourth pass). Remaining gaps: §12.14 Amp N (talkback), §12.19 Amp U
(VU drive), §12.21 cassette PSU variants, §12.30 inject. None block
Recipe #31 build.

## Recipe #32 — Neve B104A Mic Preamp (Helios / pre-1073 era)

**Era / use.** Late 1960s through early 1970s. Neve's pre-80-series
mic-preamp module, used in 1064 / 1066 consoles and the early Helios
console builds before Neve's all-silicon 1073 era took over. **Distinct
from the EMI TG12345 mic preamp** despite both being British / both
germanium-thread / both ±20-class rail. Chandler Limited's modern
"Germanium Pre" product is a direct clone of B104A, NOT of the TG12345
mic amp.

**Era recordings.** Helios was the dominant studio console at Olympic
(London) and Island Studios from ~1970 — used on Led Zeppelin III/IV,
Black Sabbath debut/Paranoid, T. Rex *Electric Warrior*, Free, Stones
*Sticky Fingers* sessions, Wings *Band on the Run* (Ginger Baker's ARC
in Lagos was Helios). When the prompt is "Helios mic preamp,"
"Olympic London 1971," or "Sabbath drum sound," this is the recipe.

**Canonical signal chain (per channel):**

```
mic input (200 Ω source via Marinair input transformer)
  → B104A module (silicon-input + germanium-driver + Si-power output):
      • Q1 = BC184C (silicon NPN input)
      • Q2 = AC176 (germanium NPN driver — not PNP like the TG12345's
              ACY21; B104A puts the Ge in the collector path of a
              high-current driver stage, not the output emitter follower)
      • Q3 = MJ2955 (silicon power BJT, output stage)
  → Marinair output transformer (VTB9049 in modern Chandler clone)
  → +28 V single-rail (NOT ±20 V split like TG12345)
```

**Front-panel controls (Chandler modern impl):**

- MIC / DI switch (DI input network: 470 pF + 15 K + 4U7 + 3K3 to a
  common front-end node)
- **THICK** switch (modifies feedback for warmer, more compressed
  character — the most-recognisable Chandler "feature" beyond the
  base Neve clone)
- Master gain pot: 6R8 + 2200 μF/50V (drives the input transformer's
  primary loading and overall feedback ratio)
- Feedback pot: 4K7 (sets module's overall NFB depth → effective gain)

**Op composition (catalog vocabulary):**

```
xformerSat[preset='marinair']
  → neveB104A (#204) [thick=false, gainDb=40]
  → germaniumBuffer (#203) [preset='neveB104A']    # if separate stage
  → xformerSat[preset='vtb9049Output']
  → gain (#1)                                       # makeup if needed
```

**Note on stage separation.** The B104A's silicon-Ge-Si chain is
internal to the module — for the recipe agent, treating
`neveB104A` (#204) as a single composite op is sufficient. The
explicit `germaniumBuffer` chain entry is only needed if the agent
needs to expose the Ge-driver bias point as a user-tunable parameter
("more germanium" knob).

**Primary source.** Chandler Germanium Pre schematic, explicitly
labeled "COPY OF NEVE B104A" — `docs/primary_sources/abbey_road/germanium/Chandler Germanium Pre/Germanium Pre.pdf`.
Canon file: `memory/emi_tg12345_console.md` §10d.6.

**Key character notes.**

- **Higher rail than TG12345.** +28 V single-rail vs ±20 V split.
  Headroom advantage: B104A clips ~+24 dBV vs TG12345's +20 dBV.
  More punch, less natural compression at high SPL.
- **Marinair transformer character.** Different LF saturation
  spectrum from EMI's BIBA-class output xfmr — Marinair's silicon-
  steel core saturates with more 3rd harmonic, BIBA leans 2nd. This
  is why B104A-derived recordings sound "tighter" and TG12345-
  derived sounds "rounder" all else equal.
- **THICK switch is the modern signature.** No equivalent on the
  vintage Neve B104A. Chandler added it to give engineers a way to
  push the module into compression for the colour at low input levels.
- **Germanium driver, not germanium output.** The Ge transistor in
  B104A is mid-chain (collector driver), not the final output stage.
  So the Ge colour is at lower signal level than in the TG's Amp E /
  Amp R (where the Ge transistor sees full output swing). Net effect:
  B104A is more *transparent* than TG12345 at moderate levels and
  about the same at peak swing.
- **No multi-stage germanium thread.** Unlike TG12345 (which has Ge
  output in Amp E + Amp G + Amp R = 3 cumulative Ge stages per
  channel), B104A is a single discrete module — only one Ge-influenced
  stage per channel. Less cumulative warmth, more clarity.

**Notes for the agent.**

- For "Led Zeppelin / Bonham drum sound": Helios console = many
  B104A's into a Helios summing buss (different topology from TG —
  Helios was a Robinson-design with its own buss-amp character).
  Closest catalog: `neveB104A` (#204) → `xformerSat[marinair]` →
  `germaniumBuffer[robinson1970]` for the buss buffer chain.
- For "Sabbath debut album guitar": guitar amp + B104A on the room
  mic. THICK on, +6 dB feedback, slight saturation into the Marinair.
- For "modern Chandler mic preamp" prompts: recipe #32 directly,
  with THICK as a user-toggleable parameter.
- For "Beatles era 1968-1969 Abbey Road tracking" prompts: this is
  the WRONG recipe — use Recipe #31 TG12345 channel strip. Abbey
  Road did NOT use Neve until the 80-series consoles arrived in the
  late 70s.

**Cross-reference.** Recipe #31 (Abbey Road TG12345) and Recipe #32
(Neve B104A / Helios) are the two pillars of British 1968–1972 mic-
preamp character. Both germanium-era, both ±20-class rails, both
discrete-BJT — but different topology, different transformers,
different cumulative-Ge structure, different recordings. **Don't
substitute one for the other when the user names a specific
artist/studio/era — the differences are audibly distinct.**
